# Bengaluru Fitness Centre Market Analysis

## Project Overview
This project analyses fitness centres across Bengaluru to identify business opportunities for launching a fitness-related service targeting students and working professionals aged 20–40.

---

## Dataset Information
- Total Records: 500
- City: Bengaluru
- Categories Covered:
  - Gym / Fitness Studio
  - Yoga Centre
  - Dance / Zumba Studio
  - CrossFit Box
  - Pilates Studio
  - Karate Dojo
  - Taekwondo Academy
  - Kickboxing Studio

---

## Analysis Performed
- Rating Distribution Analysis
- Category-wise Analysis
- Top Localities Analysis
- Review vs Rating Correlation

---

## Key Insights
- Gym and Fitness Studios dominate the Bengaluru market.
- Areas like Indiranagar and Koramangala show high fitness centre density.
- Higher review counts generally correlate with better ratings.
- Pilates and CrossFit categories show future growth potential.

---

## Final Recommendation
Indiranagar, Whitefield, and Koramangala appear to be strong market-entry opportunities due to high urban professional density and fitness awareness.
